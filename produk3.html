Belajar HTML & CSS

Sebagai Web Developer kita harus memahami terlebih dahulu tentang dasar-dasar HTML dan CSS!

Apa sih itu HTML?


HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat halaman web. HTML bekerja dengan menyusun elemen-elemen seperti teks, gambar, link, dan struktur lainnya di dalam halaman web.

Apa saja elemen di HTML: Apa saja elemen di HTML:
<header> | Menyimpan bagian kepala situs, biasanya berisi logo atau judul.
<nav> | Navigasi halaman web.
<section> | Bagian konten utama.
<article> | Konten mandiri seperti berita atau blog.
<aside> | Sidebar atau informasi tambahan.
<footer> | Bagian bawah halaman, biasanya untuk hak cipta atau kontak.

HTML menjadi fondasi dari setiap halaman web di dunia.

Apa itu CSS?

CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan dan gaya dari elemen-elemen HTML. Dengan CSS, kamu bisa mengatur warna, ukuran, tata letak, bahkan animasi pada elemen web.

Contoh penggunaan CSS:
p { color: blue; font-size: 16px; }

CSS bisa ditulis langsung di HTML (internal) atau di file terpisah seperti style.css (external).
